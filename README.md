# EPORT SOLUTIONS LIMITED - Official Website

This is the official website for **EPORT SOLUTIONS LIMITED**, a leading tech firm based in Tanzania. We specialize in software development, ICDV systems, and comprehensive digital solutions.

## 🚀 About the Project
This project is a high-performance, maintainable web application built with **Laravel 11**. It transitions from a static HTML template to a dynamic, database-driven platform to support EPORT's growing business needs.

### Key Features
- **ICDV System Integration**: Dedicated sections for ICD Management, Delivery Systems, and Container Tracking applications.
- **Dynamic Contact System**: A database-backed contact form for capturing customer inquiries directly.
- **Privacy Compliance**: Integrated Privacy Policy specifically tailored for the "EPORT DELIVERY APPLICATION" to meet Google Play Store requirements.
- **Master Layout Architecture**: Optimized for maintenance with a centralized navbar, footer, and core layouts using Blade components.
- **Responsive Design**: Fully responsive UI built with Tailwind CSS, ensuring a seamless experience across all devices.

## 🛠 Tech Stack
- **Framework**: [Laravel 11](https://laravel.com/)
- **Frontend**: [Tailwind CSS](https://tailwindcss.com/), Blade Templating
- **Database**: MySQL / MariaDB
- **PHP Version**: 8.2+

## ⚙️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone [repository-url]
   cd eport_website
   ```

2. **Install Dependencies**:
   ```bash
   composer install
   npm install && npm run build
   ```

3. **Environment Setup**:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Database Configuration**:
   Update your `.env` file with your database credentials and run migrations:
   ```bash
   php artisan migrate
   ```

5. **Run the Application**:
   ```bash
   php artisan serve
   ```
